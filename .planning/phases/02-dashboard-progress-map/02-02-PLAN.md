    **src/app/(app)/dashboard/page.tsx** — Replace stub entirely. Async Server Component:

    1. Auth + membership guard:
       ```typescript
       const supabase = await createClient();
       const { data: { user } } = await supabase.auth.getUser();
       if (!user) redirect("/login");
       ```
       Then fetch wedding using `.maybeSingle()` (NOT `.single()` — avoids throw on no-wedding case):
       ```typescript
       const { data: wedding } = await supabase
         .from("weddings")
         .select("id, name, total_budget, dismissed_welcome")
         .maybeSingle();
       if (!wedding) redirect("/onboarding");
       ```

    2. Parallel fetch milestones + budget categories:
       ```typescript
       const [milestonesRes, categoriesRes] = await Promise.all([
         supabase.from("milestones").select("*").order("sort_order", { ascending: true }),
         supabase.from("budget_categories").select("allocated_amount, expenses(amount, status)"),
       ]);
       const milestones = milestonesRes.data ?? [];
       const categories = categoriesRes.data ?? [];
       ```

    3. Compute budget totals (pure JS — no SQL aggregation):
       ```typescript
       const totalBudget = wedding.total_budget ?? 0;
       const allocated = categories.reduce((sum, c) => sum + (c.allocated_amount ?? 0), 0);
       const spent = categories
         .flatMap((c) => (c.expenses as Array<{ amount: number; status: string }> ?? []))
         .filter((e) => e.status === "paid")
         .reduce((sum, e) => sum + (e.amount ?? 0), 0);
       const remaining = totalBudget - spent;
       const budgetEmpty = totalBudget === 0;
       ```

    4. Layout structure:
       - If `!wedding.dismissed_welcome`: render `<WelcomeBanner />` at top
       - `<h1>` with wedding.name, font-serif class, text-3xl, mb-6
       - Budget row: 4 `<Card>` components in a `grid grid-cols-2 lg:grid-cols-4 gap-4 mb-8`. Each card shows label (Total Budget / Allocated / Spent / Remaining) + formatted dollar value. When `budgetEmpty`, apply `opacity-60` to the budget section and show a "Set in Settings →" link (href="/settings") below the cards.
       - Format currency: `new Intl.NumberFormat('en-US', { style: 'currency', currency: 'USD', minimumFractionDigits: 0 }).format(value)` — shows `$0` for zeros.
       - After budget row, add a `{/* MilestoneGrid placeholder */}` comment — MilestoneGrid is wired in Plan 03.

    Do NOT import MilestoneGrid yet — it doesn't exist. The comment placeholder is sufficient for Plan 02.
